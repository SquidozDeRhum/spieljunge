# Spieljunge

Spieljunge, literally 'Game Boy' in German is a Game Boy emulator which I develop from scratch.

It makes me learn a lot about architecture and is very fun to develop.

---

This project uses DearImGui and rlImGui as submodules

To clone the repo with submodules
```bash
git clone --recurse-submodules <url>
```

To get submodules in existing repo
```bash
git submodule update --init --recursive < rl>
```
